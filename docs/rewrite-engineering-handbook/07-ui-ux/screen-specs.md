# Screen Specs

## Create/Join Screen

Required controls:

- nickname input
- create room button
- room code input
- join room button

States:

- idle
- connecting
- invalid room code
- room full

## Lobby Screen

Required elements:

- room code
- copy button
- player roster
- ready toggle
- host start button
- host kick controls

Start button explains disabled reason.

## Game Screen

Required regions:

- phase header
- timer
- question
- player list
- action panel
- revealed answers
- discussion messages
- vote panel

Action panel changes by phase.

## Settlement Screen

Required elements:

- winner side
- frozen player
- all roles
- final decision vote table
- return to lobby/new game action can be P1
