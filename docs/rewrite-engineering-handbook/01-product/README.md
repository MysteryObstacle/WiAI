This directory defines what WiAI is, who it serves, which game flows are in scope, and how product behavior is accepted.

## Update Rules

- Gameplay rule changes must update [prd.md](./prd.md), [acceptance-criteria.md](./acceptance-criteria.md), and the relevant user stories.
- Flow changes must update [product-flow-diagrams.md](./product-flow-diagrams.md).
- New product constraints must be reflected in sprint acceptance criteria before development starts.
